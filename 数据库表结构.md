# 数据库表结构

> 数据库文件：`db/personal_chief.db`（SQLite）
> 最后更新：2026-05-06

---

## 1. favorites（收藏食谱表）

记录用户收藏的食谱。

```sql
CREATE TABLE IF NOT EXISTS favorites (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    thread_id       TEXT NOT NULL,
    recipe_name     TEXT NOT NULL,
    recipe_description TEXT,
    recipe_image    TEXT,
    score           REAL,
    reason          TEXT,
    ingredients     TEXT,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

| 字段 | 类型 | 说明 | 对应 Pydantic 模型 |
|------|------|------|-------------------|
| id | INTEGER | 主键，自增 | `FavoriteResponse.id` |
| thread_id | TEXT | 会话ID | `FavoriteCreate.thread_id` / `FavoriteResponse.thread_id` |
| recipe_name | TEXT | 食谱名称 | `FavoriteCreate.recipe_name` / `FavoriteResponse.recipe_name` |
| recipe_description | TEXT | 食谱描述 | `FavoriteCreate.recipe_description` / `FavoriteResponse.recipe_description` |
| recipe_image | TEXT | 食谱图片URL | `FavoriteCreate.recipe_image` / `FavoriteResponse.recipe_image` |
| score | REAL | 评分 | `FavoriteCreate.score` / `FavoriteResponse.score` |
| reason | TEXT | 推荐理由 | `FavoriteCreate.reason` / `FavoriteResponse.reason` |
| ingredients | TEXT | 所需食材（逗号分隔） | `FavoriteCreate.ingredients` / `FavoriteResponse.ingredients` |
| created_at | TIMESTAMP | 收藏时间，默认当前时间 | `FavoriteResponse.created_at` |

**对应模型定义（`app/models/schemas.py`）：**
- `FavoriteCreate` — 收藏请求（不含 id、created_at）
- `FavoriteResponse` — 收藏响应（含 id、created_at）

## 2. shopping_list（购物清单表）

记录用户需要购买的食材清单。

```sql
CREATE TABLE IF NOT EXISTS shopping_list (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    thread_id       TEXT NOT NULL,
    ingredient_name TEXT NOT NULL,
    quantity        TEXT,
    is_checked      INTEGER DEFAULT 0,
    source_recipe   TEXT,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

| 字段 | 类型 | 说明 | 对应 Pydantic 模型 |
|------|------|------|-------------------|
| id | INTEGER | 主键，自增 | `ShoppingItemResponse.id` |
| thread_id | TEXT | 会话ID | `ShoppingListGenerateRequest.thread_id` / `ShoppingItemResponse.thread_id` |
| ingredient_name | TEXT | 食材名称 | `ShoppingItemResponse.ingredient_name` |
| quantity | TEXT | 数量（如"2个"、"500g"） | `ShoppingItemResponse.quantity` |
| is_checked | INTEGER | 是否已购买（0=未买，1=已买） | `ShoppingItemResponse.is_checked` |
| source_recipe | TEXT | 来源食谱名称 | `ShoppingItemResponse.source_recipe` |
| created_at | TIMESTAMP | 添加时间，默认当前时间 | `ShoppingItemResponse.created_at` |

**对应模型定义（`app/models/schemas.py`）：**
- `ShoppingListGenerateRequest` — 生成购物清单请求（含 thread_id + items 列表）
- `ShoppingItemResponse` — 购物清单项响应（含所有字段）

## 3. 对话记忆表（LangGraph 自动管理）

由 `langgraph.checkpoint.sqlite.SqliteSaver` 自动创建和管理，用于存储对话历史。

主要表包括：
- `checkpoints` — 检查点（对话状态快照）
- `checkpoint_writes` — 检查点写入记录
- `checkpoint_blobs` — 检查点二进制数据

> 这些表由 LangGraph 框架自动维护，无需手动操作。
